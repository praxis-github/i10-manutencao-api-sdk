## Instalação

No composer.json de sua aplicação, cole o código abaixo na chave "repositories"

```
{
    "repositories" : [
        {
            "type": "vcs",
            "url": "https://github.com/praxis-github/i10-manutencao-api-sdk"
        }
    ],
    "require": {
        "praxis-github/i10-manutencao-api-sdk": "dev-master"        
    }
}

```
